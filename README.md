# NPP on Kate

NPP on Kate (Notepad++ on Kate) is a cross-platform set of scripts and plugins which aim to "port"/recreate several of the plugins and functions available in [Notepad++](https://notepad-plus-plus.org) to [Kate](https://kate-editor.org).

## Installation

Copy `npp.js` into one of the below paths depending on which platform you are on and on whether or not you want to install the plugin for the current user or all users.

```
~/.local/share/katepart5/script/commands
/usr/local/share/katepart5/script/commands
/usr/share/katepart5/script/commands
%LOCALAPPDATA%\katepart5\script\commands
%PROGRAMDATA%\katepart5\script\commands
~/Library/Application Support/katepart5/script/commands
/Library/Application Support/katepart5/script/commands
```
